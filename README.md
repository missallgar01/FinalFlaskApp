# FinalFlaskApp

A modular Flask web application demonstrating user handling and integration with external data sources.

## Overview
This project showcases a structured Flask application built with extensibility and maintainability in mind. It was developed to explore common backend patterns such as routing, modular features, and configuration management.

## Key Features
- Modular Flask application structure
- User management foundations
- Integration with external data (e.g. weather or mapping services)
- Configurable application settings

## Tech Stack
- Python
- Flask
- HTML / CSS
- REST-style routing
